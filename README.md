# Angel Manager AI Enterprise — Version 6

A one-package school management system for Angel Memorial Adventist International Academy.

## Login
- Username: `admin`
- Password: `angel123`

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to existing Streamlit app
Replace the files in your GitHub repository with this package and commit to the `main` branch. Streamlit Community Cloud redeploys the same website automatically.

Included modules: admissions, student profiles, fees and receipts, attendance, academics and grades, timetable, staff and HR, payroll, expenses, library, transport, clinic, boarding, meals, communication, portal users, construction, inventory, AI summaries, backups, and settings.
